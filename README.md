# RFSD dataset

Скрипт написан для дипломной работы. Он обрабатывает данные из базы бухгалтерской отчётности [RFSD](https://huggingface.co/datasets/irlspbru/RFSD) и собирает итоговый датасет на уровне регион-год.

Для каждого региона и года считается количество фирм, их выручка, а также отдельно те же показатели для высокотехнологичного бизнеса. Высокотехнологичные фирмы отбирались по ОКВЭД: 20, 21, 26, 27, 28, 29, 30, 32.5, 33, 50, 51, 61, 62, 63, 64, 65, 66, 69, 70, 71, 72, 75, 78, 85, 86, 87, 88.

Из данных исключены фирмы без кода региона, а также регионы 9900 (Байконур), 9000 (Запорожье), 9300 (Донецк), 9400 (Луганск), 9500 (Херсон) — по ним отсутствуют данные за значительную часть рассматриваемого периода.

## Описание переменных

| Колонка | Описание |
|---|---|
| `year` | Год |
| `region_taxcode` | Код региона ФНС |
| `region_name` | Название региона |
| `firms_n` | Общее количество фирм |
| `firms_revenue_pos_n` | Фирмы с положительной выручкой |
| `firms_revenue_zero_n` | Фирмы с нулевой выручкой |
| `firms_revenue_null_n` | Фирмы без данных по выручке |
| `firms_revenue_neg_n` | Фирмы с отрицательной выручкой |
| `ht_firms_n` | Общее количество высокотехнологичных фирм |
| `firms_revenue_ht_pos_n` | Высокотехнологичные фирмы с положительной выручкой |
| `firms_revenue_ht_zero_n` | Высокотехнологичные фирмы с нулевой выручкой |
| `firms_revenue_ht_null_n` | Высокотехнологичные фирмы без данных по выручке |
| `firms_revenue_ht_neg_n` | Высокотехнологичные фирмы с отрицательной выручкой |
| `revenue` | Суммарная выручка региона (тыс. руб.) |
| `mean_revenue` | Средняя выручка фирмы (тыс. руб.) |
| `median_revenue` | Медианная выручка фирмы (тыс. руб.) |
| `ht_revenue` | Суммарная выручка высокотехнологичных фирм (тыс. руб.) |
| `mean_ht_revenue` | Средняя выручка высокотехнологичной фирмы (тыс. руб.) |
| `median_ht_revenue` | Медианная выручка высокотехнологичной фирмы (тыс. руб.) |
| `revenue_missing_share` | Доля фирм без данных по выручке |
| `ht_revenue_missing_share` | Доля высокотехнологичных фирм без данных по выручке |
