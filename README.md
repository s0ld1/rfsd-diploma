# RFSD Region-Year Dataset Builder

Скрипт для формирования регионально-годовой панели высокотехнологичных фирм на основе базы бухгалтерской отчётности [RFSD](https://huggingface.co/datasets/irlspbru/RFSD).

## Что делает скрипт

Из исходных данных RFSD формирует CSV-файл с агрегированными показателями по каждому региону РФ за каждый год (2011–2024):

- Количество фирм и их распределение по статусу выручки
- Суммарная, средняя и медианная выручка по региону
- Аналогичные показатели отдельно для высокотехнологичных фирм
- Диагностические показатели качества данных

## Высокотехнологичные отрасли (ОКВЭД)

Фирма относится к высокотехнологичному сектору, если её основной ОКВЭД относится к одному из следующих разделов:

`20, 21, 26, 27, 28, 29, 30, 32.5, 33, 50, 51, 61, 62, 63, 64, 65, 66, 69, 70, 71, 72, 75, 78, 85, 86, 87, 88`

## Очистка данных

- Исключены фирмы без кода региона (`region_taxcode IS NULL`)
- Исключены регионы: 9900 (Байконур), 9000 (Запорожье), 9300 (Донецк), 9400 (Луганск), 9500 (Херсон)
- Выручка суммируется только для фирм с неотрицательной выручкой (`>= 0`)
- Средние и медианные значения считаются только по фирмам с положительной выручкой (`> 0`)

## Зависимости

```
pip install duckdb
```

## Запуск

Укажите путь к папке с parquet-файлами RFSD и путь для сохранения результата непосредственно в скрипте (`DATA_DIR` и `OUT_CSV`), затем:

```bash
python rfsd_build.py
```

## Выходные колонки

| Колонка | Описание |
|---|---|
| `year` | Год |
| `region_taxcode` | Код региона ФНС |
| `region_name` | Название региона |
| `firms_n` | Общее количество фирм |
| `firms_revenue_pos_n` | Фирмы с положительной выручкой |
| `firms_revenue_zero_n` | Фирмы с нулевой выручкой |
| `firms_revenue_null_n` | Фирмы без данных по выручке |
| `firms_revenue_neg_n` | Фирмы с отрицательной выручкой |
| `ht_firms_n` | Общее количество высокотехнологичных фирм |
| `firms_revenue_ht_pos_n` | Высокотехнологичные фирмы с положительной выручкой |
| `firms_revenue_ht_zero_n` | Высокотехнологичные фирмы с нулевой выручкой |
| `firms_revenue_ht_null_n` | Высокотехнологичные фирмы без данных по выручке |
| `firms_revenue_ht_neg_n` | Высокотехнологичные фирмы с отрицательной выручкой |
| `revenue` | Суммарная выручка региона (тыс. руб.) |
| `mean_revenue` | Средняя выручка фирмы (тыс. руб.) |
| `median_revenue` | Медианная выручка фирмы (тыс. руб.) |
| `ht_revenue` | Суммарная выручка HT-фирм (тыс. руб.) |
| `mean_ht_revenue` | Средняя выручка HT-фирмы (тыс. руб.) |
| `median_ht_revenue` | Медианная выручка HT-фирмы (тыс. руб.) |
| `revenue_missing_share` | Доля фирм без данных по выручке |
| `ht_revenue_missing_share` | Доля HT-фирм без данных по выручке |
